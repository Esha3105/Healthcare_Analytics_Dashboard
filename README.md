# Healthcare Analytics Dashboard

## 📊 Project Overview

This project analyzes hospital operations and financial performance using SQL and Power BI. It focuses on patient trends, billing, insurance contributions, and pending payments to support data-driven decision-making.

The project simulates real-world healthcare analytics using multiple relational tables.

---

## 🎯 Objectives

* Analyze patient volume and repeat visits
* Track hospital revenue and insurance payments
* Identify pending payments (revenue leakage)
* Understand monthly patient trends
* Compare performance across departments and payment methods

---

## 🛠️ Tools & Technologies

* Power BI – Dashboard creation and visualization
* SQL (MySQL) – Data analysis and querying
* Excel – Data cleaning and preprocessing

---

## 📁 Dataset

The dataset consists of multiple related healthcare tables:

* Patients
* Appointments
* Billing
* Doctors
* Treatments

---

## 🧠 SQL Analysis (Multi-Table Joins)

Performed **multi-table joins** to combine data from different sources and generate meaningful insights.

### 🔹 Joins Used

* INNER JOIN
* LEFT JOIN

### 🔹 Example Use Cases

* Linked **patients with appointments** to track visit frequency
* Combined **billing and insurance tables** to analyze payment distribution
* Joined **treatment and specialization data** to evaluate cost by department
* Connected multiple tables to calculate **pending vs completed payments**

### 🔹 SQL Techniques Applied

* JOINS (multi-table queries)
* GROUP BY & Aggregations
* CASE WHEN for categorization
* Subqueries for advanced filtering
* KPI calculations (totals, averages, percentages)

👉 This step enabled creation of structured datasets for dashboard reporting.
